# Проект нацелен на скрытие и сортировку банковских счетов и карт.
 -файл masks.py скрывает банковские счета и банковские карты.
 -файл processing.py предназначен для фильтрации и сортировки данных.

## Пример работы функции:
`def get_mask_card_number(card)` - маскирует банковскую карту. Пример: 7000 79** **** 6361
`def get_mask_account(card)` - маскирует банковские счета. Пример: ** 4305
`def mask_account_card(info: str) -> str:` - максирует банковские счета и карты, работая с функциями выше, но оставляя названия платёжных систем.Пример: Visa Platinum 7000 79** **** 6361 Счет **4305
`def get_date(date_string: str)` - формирует дату в формат ДД.ММ.ГГГГ. Пример: 04.01.2026
`def filter_by_state(transactions, state='EXECUTED')` - сортировка по значению ключа. Пример: [{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}]
[{'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}, {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}]
`def sort_by_date(transactions, descending=True)` - сортировка по дате. Пример:  [{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}, 
{'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}, {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}]
